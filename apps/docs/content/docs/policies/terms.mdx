## Service Description

Documenso is a document signing platform that enables you to:

- Upload PDF documents for signature
- Add signature fields and other form fields
- Send documents to recipients for signing
- Track document status and completion
- Generate audit trails for signed documents
- Store completed documents

Documenso is available as a cloud-hosted service (Documenso Cloud) and as self-hosted software that you can run on your own infrastructure.
