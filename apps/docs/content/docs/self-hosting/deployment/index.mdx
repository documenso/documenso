# Deployment

Choose a deployment method based on your needs.

| Method                                                         | Best For                      | Difficulty |
| -------------------------------------------------------------- | ----------------------------- | ---------- |
| [Docker](/docs/self-hosting/deployment/docker)                 | Development, testing          | Easy       |
| [Docker Compose](/docs/self-hosting/deployment/docker-compose) | Production, small teams       | Easy       |
| [Railway](/docs/self-hosting/deployment/railway)               | Quick deployment, managed     | Very Easy  |
| [Kubernetes](/docs/self-hosting/deployment/kubernetes)         | Enterprise, high availability | Advanced   |
