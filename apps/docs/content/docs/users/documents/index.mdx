## Document Workflow

1. **[Upload](/docs/users/documents/upload)** - Upload your PDF and configure settings
2. **[Add Recipients](/docs/users/documents/add-recipients)** - Specify who needs to sign, approve, or view
3. **[Add Fields](/docs/users/documents/add-fields)** - Place signature, text, and other fields
4. **[Send](/docs/users/documents/send)** - Send for signing and monitor progress
