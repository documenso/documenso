## Quick Start

Get started with the Documenso API in minutes:

### 1. Get Your API Key

Generate an API token from your [Documenso dashboard](https://app.documenso.com/settings/tokens).

### 2. Make Your First Request

```bash
curl -X GET "https://app.documenso.com/api/v1/documents" \
  -H "Authorization: YOUR_API_TOKEN"
```

### 3. Create a Document

```bash
curl -X POST "https://app.documenso.com/api/v1/documents" \
  -H "Authorization: YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "My Document",
    "recipients": [
      {
        "email": "signer@example.com",
        "name": "John Doe",
        "role": "SIGNER"
      }
    ]
  }'
```

For detailed API documentation, see the [API Reference](/docs/developers/public-api/reference).
