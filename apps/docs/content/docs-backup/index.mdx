## What is Documenso?

Documenso is the open source DocuSign alternative. It provides a modern, transparent, and secure way to sign documents electronically.

### Key Features

- **Open Source** - Full transparency with community-driven development
- **Self-Hostable** - Deploy on your own infrastructure for complete data sovereignty
- **API-First** - Comprehensive REST API for integrations
- **Templates** - Create reusable document templates
- **Team Collaboration** - Work together with shared documents and workflows
- **Legally Binding** - eIDAS and ESIGN compliant electronic signatures
